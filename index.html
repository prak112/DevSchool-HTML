Embedding Images, Video and Subtitles

In my view, this is the best illustration of a T-Rex relatable to the current world. Also helps in viewing and understanding the T-Rex out of curiosity. Set up is as follows :

  • Create an index.html file and code with guidance from MDN Web Docs
  • Find an image of your interest from Freepik
  • Add image to index.html (FreePik is an open source library)
  • Create a styles.css file to set colors, any text (like this)
  • Run it on a local server or just open with Chrome/Edge.

This is a simple MP4 video of a cute koala eating leaves from Pexels by Valeriia Miller. The poster image is a koala-generated-by-ai from FreePik by vecstock. Set up is as follows :

  • Search for free and permissive videos to host in your page (Found it on Pexels)
  • Add a new <video> element with source and controls
  • Remember to credit the source and the author, as I did above.
  • Add a poster image before the video
  • Voila! The video is hosted on your page!
  • Coincidentally, it is the same procedure for audio clips as well.
