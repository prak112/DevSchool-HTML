            Coming to "How to set this up" is as follows :
            - Create a new Orphan branch in your Git repo
            - Create .gitignore with basic ignore rules from Toptal
            - Create a README.md file, to write a one-line description, helps to be clear about what exactly to do.
            - Create an index.html file and code with guidance from MDN Web Docs
            - Find an image of your interest from Freepik and add it to index.html
            - Create a styles.css file to set colors, any text (like this)
            - Run it on a local server or just debug using Chrome/Edge
            - Voila! Here is what you get!
