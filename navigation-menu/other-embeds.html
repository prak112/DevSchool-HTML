Different types of embedding elements include :

  • <iframe> - Embeds an external web page
  • <embed> - Embeds an external media file
  • <object> - Embeds an external application

  • However, the most common and popular are : <iframe>, <canvas>, <video> and so on.
  • <embed> & <object> are deprecated due to their lack of support in modern browsers considering the security and accessibility risks from external resources.

Takeaways from Embedding

Source : MDN Web Docs

  • Embed only when and where it is needed, to prevent possible attacks since it is not secure.
  • One of the possible attacks is Clickjacking
  • To prevent Clickjacking attacks, do the following :
  • Always use sandbox attribute.
  • Use HTTPS-enabled hosting with neccessary certification using Let's Encrypt.
  • Secure all <iframe> elements by configuring server in JS/HTTP/XML to send appropriate X-Frame-Options header.
