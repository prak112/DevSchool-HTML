Issues which provided the neccessity for Responsive Design or in our case, Responsive Images :

  • Art Direction Problem - Rendering images specific for browser/device size
  • Resolution Switching Problem - Same images to be displayed in differing number of pixels based on browser/device size

Surprisingly, the solution to either of the problems mentioned above is to add images of different resolutions, detail and orientation as follows :

  • Resolution Switching Problem - Use <img> element and its srcset and sizes attributes, OR SVG images if not too complex
  • Art Direction Problem - Crop images as required for different layouts and use <picture> and <source> elements with media and srcset attributes.
